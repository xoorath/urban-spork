# urban-spork

Requires:
- [node.js](https://nodejs.org/en/) 
- [yarn](https://yarnpkg.com/en/docs/getting-started)
- [visual studio](https://www.visualstudio.com/downloads/)

Start:
- yarn install
- yarn run setup
- open `.build/projects/vs2017/urbanspork.sln` in visual studio.
